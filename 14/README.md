## 14.2 进入32位保护模式

### 14.2.2 创建GDT并安装段描述符

```s
pdgt dw 0
     dd 0x00007e00
```

* 在32位处理器上，即使是在实模式下，也可以使用32位寄存器
* 如果需要访问代码段内的数据，只能重新为该段安装一个新的描述符，并将其定义为可读可写的数据段
